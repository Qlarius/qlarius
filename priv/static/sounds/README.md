# Sound Assets

## Required Files

### coin-clink.mp3
A short coin/clink sound (~200-500ms) that plays when wallet balance increases.

**Suggested sources for free sounds:**
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/search/coin/) (royalty-free)
- [Freesound.org](https://freesound.org/search/?q=coin+clink) (check individual licenses)
- [Mixkit](https://mixkit.co/free-sound-effects/coin/) (royalty-free)

Place the file as `coin-clink.mp3` in this directory.
