# The Daily Local - Demo News Site

A mockup of a traditional local news website for Springfield, designed to demonstrate embedded iframe widgets.

## Structure

- **index.html** - Main homepage with responsive layout
- **assets/** - Images and graphics
  - `story-*.svg` - Story illustration placeholders

## Features

- Traditional newspaper design with bold Cinzel Roman capitals masthead and Merriweather body text
- Responsive layout (mobile to desktop)
- Navigation sections: Local News, Sports, Business, Opinion, Arts & Culture, Weather
- Homepage sections:
  - Featured/Breaking news story
  - Local news grid
  - Sports highlights
  - Business updates
  - Sidebar with weather, opinion pieces, events, and most-read

## Styling

- Uses Tailwind CSS via CDN
- Masthead font: Cinzel (Google Fonts) - bold Roman capitals, authoritative newspaper style
- Primary font: Merriweather (Google Fonts) - classic newspaper serif
- Secondary font: System sans-serif for UI elements
- Traditional news color scheme: black, white, grays with blue accents

## Access

Local: `priv/static/demosite/local_news/index.html`
Server: `/demosite/local_news/` (when Phoenix is running)
